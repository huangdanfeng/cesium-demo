# cesium-demo
  
  cesium 三维可视化
  
  **<font size=4>安装</font>**
  
  由于cesium启动需要服务器，不可以直接打开html，所以先装http-server

```javascript
  npm install http-server -g
```

  
  **<font size=4>启动</font>**
  
  进入cesium-putong文件夹，然后使用http-server启动项目:
  
  <img src="/image/localhost.png"  width="600" height="400">
  
  最后浏览器打开http://127.0.0.1:8081/index.html
